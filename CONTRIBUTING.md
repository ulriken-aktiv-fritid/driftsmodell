# Bidra til Ulriken Aktiv Fritid - Driftsmodell

Takk for at du ønsker å bidra! Vi setter pris på all hjelp for å forbedre nettsiden.

## Hvordan bidra

### Rapportere feil
- Bruk issue-malen for feilrapporter
- Gi så mye detaljer som mulig
- Inkluder skjermbilder hvis relevant

### Foreslå nye funksjoner
- Bruk issue-malen for funksjonsforespørsler
- Beskriv problemet og ønsket løsning tydelig

### Bidra med kode
1. Fork repository
2. Lag en feature-branch (`git checkout -b feature/ny-funksjon`)
3. Gjør endringene dine
4. Test endringene lokalt
5. Commit endringene (`git commit -m 'Legg til ny funksjon'`)
6. Push til branch (`git push origin feature/ny-funksjon`)
7. Åpne en Pull Request

### Kodingstil
- Bruk semantisk HTML
- Hold CSS konsistent med eksisterende stil
- Kommenter kompleks kode
- Test i flere nettlesere

### Lokale endringer
For å teste endringer lokalt:
```bash
npm run dev
# eller
python3 -m http.server 8000
```
